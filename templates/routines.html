Routine Configuration

Frequency = times per week (7 = daily, 14 = 2x/day, 0.25 = monthly). Weight (try 1–5) scales how much each task affects your dashboard score for completed scheduled dots. Check weekdays to pin a task (e.g. trash every Tuesday).

Push: only when today has a scheduled dot for that task that is still empty—not for missed dots on earlier weekdays by themselves. Clear the Notify field on a task to turn off its timed reminder.

Push notifications

Sends one test to every device that tapped Enable reminders. Notify defaults to {{ default_notify_time }} (set LM_DEFAULT_NOTIFY_TIME to change). Times use this computer’s clock. Clear a task’s time to disable its reminder.
